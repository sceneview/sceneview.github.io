Compose State drives your 3D data

SceneView nodes are just Composable functions. Connect them to your ViewModel's StateFlow and every data update re-renders instantly — no imperative scene graph manipulation, no separate rendering loop.

Real-time Data Compose State 60fps Animations
// Data updates → 3D reacts instantly val sales by viewModel.salesFlow.collectAsState() Scene { sales.forEachIndexed { i, value -> CubeNode( size = Position(0.4f, value, 0.4f), color = Color(0.1f, 0.5f, 1f) ) { position(x = i * 0.6f) } } }
